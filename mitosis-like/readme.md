# Mitosis-like

[ErgoSNM keyboard](https://github.com/siderakb/ergo-snm-keyboard) wireless [mitosis](https://github.com/reversebias/mitosis)-like edition firmware.

## MCU

nRF52840 (MDBT50Q-P1MV2)
